# v1.4

Date: 2026-06-03
Type: minor

## Summary
- Word Games release: adds Five Letter and Word Grid.

## User-facing changes
- Video Mode now has a Home screen control with an inline enable switch and placement picker; the old Settings entry point was removed.
- Five Letter: 5-letter word guessing with Daily and Unlimited modes, six guesses, keyboard feedback, resume support, stats, and Video Mode support.
- Five Letter strict mode now enforces prior green/yellow/blank feedback.
- Daily Five Letter is now a one-shot challenge: it cannot be restarted mid-attempt, remains reviewable after completion, routes players to Unlimited afterward, and reports a separate Daily Challenge win rate.
- Word Grid: 4x4 adjacent-letter word tracing with Timed and Relaxed modes, score tracking, resume support, stats, and Video Mode support.
- Word Grid now generates less-obvious boards from discoverable local word paths, expands the local word list, and supports swipe selection across adjacent tiles.
- Word Grid now keeps found words visible during play and after finishing, using a side rail on wider screens and a compact list on narrow screens.
- Home now includes Five Letter and Word Grid as playable game tiles.

## Internal changes
- Added shared word lexicon loading and bundled word resources under `Resources/Words/`.
- Added pure Five Letter feedback logic and Word Grid path/scoring logic with unit tests.
- Extended routing, game identity, reset behavior, and stats surfaces for both word games.
- Tech-debt cleanup: consolidated the triple-duplicated Solitaire/FreeCell board "felt" color into a single `SolitaireFelt` source of truth, gated `NonogramLibrary` diagnostic `print()`s behind `#if DEBUG`, and removed a redundant `nonisolated(unsafe)` to clear the only code build warning.

## Fixes
- Video Mode's large-bottom control row now sits lower to avoid the Dynamic Island, and two-mode pickers now toggle from any tap within the pill.
- Fixed Nonogram restore validation so stale or malformed 20 x 20 saves are discarded instead of reopening into a bad board state.
- Fixed Nonogram clue strike-through so a single unbounded fill no longer crosses off a `1` when it could still belong to a longer run.
- Fixed Nonogram restart after "View board" so later win/loss banners show again.
- Five Letter now accepts any 5-letter A-Z guess instead of rejecting common
  words outside the seed list.
- Word Grid now generates from guaranteed playable local boards instead of
  random letters that could produce too few valid words.

## Risks / notes
- The bundled word lists are intentionally small seed lists with provenance scaffolding; expand them from the documented SCOWL/ESDB pipeline before release screenshots or TestFlight.
- `Localizable.xcstrings` was already dirty before this implementation pass and was not regenerated here.
