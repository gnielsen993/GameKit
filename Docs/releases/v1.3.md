# v1.3

Date: 2026-05-23 → 2026-05-25
Type: minor

## Summary
- FreeCell — third game in GameDrawer. 8-column tableau, 4 free cells,
  4 foundations, multi-card sequence moves, deal-number entry, win/loss
  detection, win cascade animation, Video Mode support, and stats.
- Solitaire (Klondike) — fourth game in GameDrawer. 7-column tableau,
  stock/waste pile, 4 foundations, draw-1/2/3 difficulty, auto-complete,
  and stats.
- Home screen overhaul — 3-column icon grid replaces accordion drawer rows;
  tap to expand into horizontal strip + detail panel with mode chips.

## User-facing changes

### FreeCell
- New game accessible from the home screen.
- Three random difficulty tiers (Easy / Medium / Hard / Expert) plus
  a Deal # mode for entering a specific deal number (1–32,000).
- Tap or drag to move cards; multi-card sequences are capped by the
  number of empty free cells and empty columns (standard FreeCell
  supermove rule).
- Double-tap any top card to auto-send it to the foundation.
- Undo button in the toolbar; reset restarts the same deal.
- Loss detection — game ends if no legal moves remain.
- Win cascade animation: all foundation cards spread across the screen
  on victory. Win haptic (success) + brief green flash overlay fire at
  the moment the board is solved.
- Video Mode: reflows for all six PiP zones (large top/bottom + four
  corners).
- Stats card in StatsView: games, wins, win %, and best time per
  difficulty tier (Easy / Medium / Hard / Expert / Deal #).

### Solitaire
- New Klondike Solitaire accessible from the home screen.
- Three difficulty modes: Easy (Draw 1), Medium (Draw 2), Hard (Draw 3).
- Tap to select/move; drag cards or sequences between columns.
- Drag top waste card onto tableau columns.
- Double-tap any top tableau card or waste card to auto-send to
  foundation.
- Auto-Complete button appears when all remaining cards can be swept to
  foundations automatically.
- Undo button (up to 50 moves); restart-this-deal available in the
  toolbar menu.
- Win banner shows move count with a new-game prompt.
- Win haptic (success) + brief green flash overlay fire at the moment of victory
  (auto-complete and manual both trigger it).
- Stats card in StatsView: wins, win %, and best time per difficulty.
- Deal number shown in the header for reference.

### Home screen
- 3-column `LazyVGrid` of `GameIconView` tiles replaces the accordion
  `DrawerRow` layout.
- Tap to expand: selected tile scales to 96pt; others collapse to 44pt in
  a horizontal `ScrollView` strip; `HomeDetailPanel` appears below with
  mode chips and a Stats link.
- Canvas-based `GameIconView` renders per-game SVG glyphs at any size
  with no image assets; `GameKind.accentColor` drives tile backgrounds.
- `DrawerRow.swift` and `DrawerChrome.swift` deleted.
- `GameDescriptor` gains `isNew: Bool` (badge) and `shortMeta: String`
  (detail panel subtitle).

## Internal changes
- Mid-game save state for all five games (Minesweeper, Nonogram, FreeCell,
  Solitaire, Merge): `{Game}SaveState: Codable` structs stored in
  UserDefaults; scenePhase `.background` auto-saves and `.active`
  resumes; resume prompt on re-entry; `GameStats.resetAll()` clears all
  save slots.
- `FreeCellViewModel`, `FreeCellBoard`, `FreeCellRules`, `FreeCellDeal`,
  `FreeCellDifficultyIndex`: pure engine + observable VM with undo
  history, timer, and haptic counter-triggers.
- `FreeCellGameView` + `FreeCellGameView+VideoMode`: normal and
  video-mode layouts; drag state managed in view layer.
- `FreeCellStatsCard`: Grid-based stats table, reads `GameRecord` /
  `BestTime` from `GameStats`.
- `SolitaireViewModel`, `SolitaireBoard`, `SolitaireRules`: pure engine
  + observable VM; history capped at 50 snapshots.
- `SolitaireGameView`: drag overlay floating in a board-level ZStack so
  it can travel over both the top row and the tableau simultaneously.
- `SolitaireStatsCard`: stats per difficulty with wins, win %, and best
  time.
- Both games record to `GameStats` via `gameKind: .freeCell` /
  `.klondike`.
- `GameKind`, `GameRoute`, `GameDescriptor`: Solitaire entries added.
- `HomeView`, `StatsView`: wired for both games.

## Fixes
- FreeCell: disabled swipe-back gesture app-wide to prevent accidental navigation mid-drag.
- FreeCell: PiP toolbar items no longer duplicate on re-entry to video mode.
- FreeCell: hint toast replaced with top-aligned overlay (no layout shift).
- FreeCell: card readability, column slot outlines, drag highlight, and top-card
  selection highlight polished.
- FreeCell: hit detection improved for multi-card drag from deep stacks.
- Home: removed stale "New" badge; two-tier mode picker fixed for all game routes;
  per-game stats cards wired from Stats tab.
- Home: auto-complete animation fires correctly after save-state restore.
