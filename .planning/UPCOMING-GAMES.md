# Upcoming Games — vetted backlog

**Created:** 2026-06-25 · Living list. Not committed scope — candidates
vetted for brand fit + spine fit, ordered roughly by readiness.

Spine fit = pure Foundation-only engine, MVVM, theme tokens, SwiftData
stats. Brand = calm / premium / ad-free.

---

## Arcade (after v1.5 proves the loop substrate)

Reuse the v1.5 real-time loop. These lean twitchier than the calm-only
v1.5 set — gate by mood, not capability.

- **Glide / Drift** *(working title — NOT "Flappy Bird")* — tap-to-rise,
  navigate gaps, score-until-death. The original idea, renamed. Twitchier
  brand fit, so it waits behind the calm pair. Strong "one more try" loop.
  Pick a clean geometric name (Glide / Drift / Updraft) over a bird theme.
- **Rhythm tile-tap** *(Piano-Tiles-like)* — tap falling tiles in time;
  calmer than Glide, theme-friendly. Could even ride the calm line.
- **Drop / falling blocks** *(Tetris-like)* — endless, beloved, but most
  "me-too" / legally cautious. Lower priority.

## Logic (off the original roadmap — your wheelhouse)

- **Flow** — connect colored pipes to fill the grid. Pure engine, infinite
  generated levels, theme-gorgeous. **Highest-value next logic game.**
- **Pattern Memory** *(Simon)* — sequence/timing; a natural bridge between
  logic and the arcade loop (shares run-until-fail shape).
- **Chess puzzles** *(mate-in-N)* — highest value, heaviest lift: needs a
  legal-move engine + curated puzzle DB. **Own milestone, not an add-on.**
- **Lights Out · Pipes/rotate · Hashi (Bridges) · Kakuro** — small,
  spine-clean logic fillers. Kakuro pairs naturally with Sudoku.
