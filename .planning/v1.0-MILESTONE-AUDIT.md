1. **First-launch flow:** Cold install → 3-step intro (themes → stats → SIWA optional) → Skip → HomeView with Mines enabled. ✓ (P5 SC3)
2. **Play-and-persist flow:** Tap Mines card → game → win/loss → GameStats writes → cold restart → Stats screen shows record. ✓ (P4 SC1+SC5)
3. **Theme-customize flow:** Settings → 5 Classic swatches inline OR navigate to FullThemePicker → palette change reaches Mines grid. ✓ (P5 SC4 + THEME-03)
4. **Stats export/import flow:** Settings DATA → Export → fileExporter → save JSON; Import → fileImporter → schema-validate → transactional replace. ✓ (P4 SC4 manual §7)
5. **Sign-in promotion flow:** Settings → SYNC → SIWA → Restart prompt → quit/relaunch → CloudKit mirroring → second device sees data. ✓ (P6 SC3)
