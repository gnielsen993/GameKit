<objective>
Land the JSON Export/Import surface that round-trips a stats snapshot through `fileExporter` / `fileImporter` byte-for-byte. Per PERSIST-03 + SC4: "Export to JSON via `fileExporter` and re-import via `fileImporter` produces a byte-for-byte round-trip including `schemaVersion` — exporting a 50-game stats set, resetting stats, re-importing produces the original counts and best times."

Purpose: Stats export is the user's only **out** of the local-only data model — `gamekit-stats-YYYY-MM-DD.json` to Files app, AirDrop, iCloud Drive, etc. Per CLAUDE.md §1 data-safety: "Implement Export/Import JSON with `schemaVersion` for stats." This plan ships the codec + transaction logic; Plan 05 wires the Settings rows that drive it.

Per RESEARCH Pitfall 6, the import flow's correctness rests on **decode-then-validate-then-transaction** ordering. Validating `schemaVersion` BEFORE opening the delete transaction is non-negotiable — the wrong order destroys existing data when a future-schema file lands. Per RESEARCH Pitfall 7, `.sortedKeys + .iso8601 + .prettyPrinted` is the load-bearing encoder configuration for SC4 byte-for-byte determinism.

    Type shape (verbatim from `04-PATTERNS.md` line 540–550 + RESEARCH §Pattern 3, expanded with full bodies):
    ```swift
    import Foundation
    import SwiftData
    import os

    @MainActor
    enum StatsExporter {

        /// Pinned at 1 in P4; bumping requires a deliberate, additive migration plan.
        static let envelopeSchemaVersion: Int = 1

        private static let logger = Logger(
            subsystem: "com.lauterstar.gamekit",
            category: "persistence"
        )

        // MARK: - Export

        static func export(modelContext: ModelContext) throws -> Data {
            let records = try modelContext.fetch(FetchDescriptor<GameRecord>())
            let bests = try modelContext.fetch(FetchDescriptor<BestTime>())

            let envelope = StatsExportEnvelope(
                schemaVersion: envelopeSchemaVersion,
                exportedAt: .now,
                gameRecords: records.map { rec in
                    StatsExportEnvelope.Record(
                        id: rec.id,
                        gameKindRaw: rec.gameKindRaw,
                        difficultyRaw: rec.difficultyRaw,
                        outcomeRaw: rec.outcomeRaw,
                        durationSeconds: rec.durationSeconds,
                        playedAt: rec.playedAt,
                        schemaVersion: rec.schemaVersion
                    )
                },
                bestTimes: bests.map { best in
                    StatsExportEnvelope.Best(
                        id: best.id,
                        gameKindRaw: best.gameKindRaw,
                        difficultyRaw: best.difficultyRaw,
                        seconds: best.seconds,
                        achievedAt: best.achievedAt,
                        schemaVersion: best.schemaVersion
                    )
                }
            )

            let encoder = JSONEncoder()
            encoder.dateEncodingStrategy = .iso8601                    // UTC, "Z" suffix
            encoder.outputFormatting = [.prettyPrinted, .sortedKeys]   // SC4 byte-for-byte
            return try encoder.encode(envelope)
        }

        // MARK: - Import (decode → validate → transaction → save)

        static func importing(_ data: Data, modelContext: ModelContext) throws {
            // Step 1: Decode FIRST. If the JSON is malformed, throw before
            //         touching SwiftData. RESEARCH Pitfall 6.
            let decoder = JSONDecoder()
            decoder.dateDecodingStrategy = .iso8601

            let envelope: StatsExportEnvelope
            do {
                envelope = try decoder.decode(StatsExportEnvelope.self, from: data)
            } catch {
                logger.error("Import decode failed: \(error.localizedDescription)")
                throw StatsImportError.decodeFailed
            }

            // Step 2: Validate schemaVersion. If mismatch, throw — existing
            //         data UNTOUCHED (no transaction opened yet).
            guard envelope.schemaVersion == envelopeSchemaVersion else {
                throw StatsImportError.schemaVersionMismatch(
                    found: envelope.schemaVersion,
                    expected: envelopeSchemaVersion
                )
            }

            // Step 3: Open replace-on-import transaction (D-20). Atomic.
            try modelContext.transaction {
                try modelContext.delete(model: GameRecord.self)
                try modelContext.delete(model: BestTime.self)

                for r in envelope.gameRecords {
                    let rec = GameRecord(
                        gameKind: GameKind(rawValue: r.gameKindRaw) ?? .minesweeper,
                        difficulty: r.difficultyRaw,
                        outcome: Outcome(rawValue: r.outcomeRaw) ?? .loss,
                        durationSeconds: r.durationSeconds,
                        playedAt: r.playedAt
                    )
                    rec.id = r.id                          // preserve original UUID for round-trip equality
                    rec.schemaVersion = r.schemaVersion
                    modelContext.insert(rec)
                }
                for b in envelope.bestTimes {
                    let best = BestTime(
                        gameKind: GameKind(rawValue: b.gameKindRaw) ?? .minesweeper,
                        difficulty: b.difficultyRaw,
                        seconds: b.seconds,
                        achievedAt: b.achievedAt
                    )
                    best.id = b.id
                    best.schemaVersion = b.schemaVersion
                    modelContext.insert(best)
                }
            }

            // Step 4: Synchronous save (RESEARCH Pitfall 10). Force-quit-after-import
            //         must immediately persist the imported set.
            try modelContext.save()
        }

        // MARK: - Filename helper (D-19)

        /// Produces "gamekit-stats-YYYY-MM-DD.json" — locale-independent ISO date.
        static func defaultExportFilename(now: Date = .now) -> String {
            let formatter = ISO8601DateFormatter()
            formatter.formatOptions = [.withFullDate]   // YYYY-MM-DD (RESEARCH A7)
            return "gamekit-stats-\(formatter.string(from: now)).json"
        }
    }
    ```

    **Critical rules (call out in file header + inline comments):**
    1. **Decode-then-validate-then-transaction** — RESEARCH Pitfall 6. Schema-mismatch detection happens BEFORE the destructive `delete(model:)` calls. Wrong order destroys existing data on a future-schema file.
    2. **`.sortedKeys` is non-negotiable** for SC4 byte-for-byte determinism — RESEARCH Pitfall 7.
    3. **UUID preservation across round-trip** — re-inserted GameRecord/BestTime have their `id` overwritten from the envelope so round-trip equality holds. Default `id: UUID = UUID()` produces fresh UUIDs without this; the assignment after `init(...)` corrects it. (The `.unique` constraint is forbidden per Pitfall 2, so duplicate UUIDs cause no failure — only round-trip equality demands preservation.)
    4. **Schema version preservation** — same reasoning; `rec.schemaVersion = r.schemaVersion` keeps the envelope's nested per-row schemaVersion intact (degenerate today since they all equal 1, but future-proof for partial migrations).
    5. **Loss path & best-time interaction**: `importing` does NOT re-evaluate BestTime via `GameStats.evaluateBestTime` — it inserts the envelope's BestTime rows verbatim. Replace-on-import semantics per D-20.
