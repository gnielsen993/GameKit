# Phase 06.1 Plan 02: Minesweeper Reveal/Flag Interaction-Mode Toggle Summary

MINES-12 satisfied. Added a Reveal/Flag interaction-mode toggle to Minesweeper. Two modes: `.reveal` (default ‚Äî tap reveals, long-press flags ‚Äî current MINES-02 contract preserved verbatim) and `.flag` (tap toggles flag, long-press reveals ‚Äî inverted). Long-press always provides a quick-action escape from the current mode. The toggle UI is a 64pt circular FAB below the board, color-coded (`theme.colors.accentPrimary` ‚Üî `theme.colors.danger`) and icon-swapped (`cursorarrow.click` ‚Üî `flag.fill`) per mode, with light haptic on toggle. Mode resets to `.reveal` on every `restart()` (no UserDefaults persistence per D-08). The 0.25s long-press threshold + `.exclusively(before:)` gesture composition are PRESERVED VERBATIM (CellView byte-identical between HEAD~3 and HEAD; verified by zero-line diff).
