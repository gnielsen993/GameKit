prominent "HIGH SCORE" hero numeral above smaller Average Score / Runs Played rows
(+ Stack-only Best Streak), with the empty state "No runs yet." Extract the shared
layout into one `ScoreStatsCard` component and shrink both game cards to
derivation-only wrappers (D-08). Append the release-log entry (CLAUDE.md §8.14).

Purpose: The arcade cards must read as a deliberately different shape class from the
turn-based win/loss/best-time column-grid cards, while staying inside GameDrawer's
visual language (StackScoreChip label idiom, DKCard hosting, token discipline).
Output: `ScoreStatsCard.swift` (new), rewritten `StackStatsCard.swift` /
`SnakeStatsCard.swift` wrappers, release-log append.

| T-18-01 | Tampering | Local GameRecord.score / BestScore rows feeding averageScoreText | accept | Device-local store, no privilege boundary crossed; this plan adds no write path (derivation only per D-06). Tampering affects only the tamperer's own displayed stats. |
