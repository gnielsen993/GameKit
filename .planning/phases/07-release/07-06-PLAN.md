## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Xcode UI ↔ project.pbxproj | Xcode writes the version + build values; CLAUDE.md §8.8 forbids hand-patching pbxproj. The diff for the version bump must show Xcode's exact format, not a hand edit. |
| Local archive ↔ TestFlight | The archive that ships to TestFlight is the FIRST binary that touches Production CloudKit (per Plan 07-03). Schema mismatch = SC3 fails on missing record types. |
| TestFlight build ↔ App Store Review | The same binary App Reviewers test must satisfy SC1-SC5. Privacy nutrition label answer must match the binary (Pitfall 11). |
| Soak window ↔ submission decision | Premature submit (no soak) = Apple Reviewer hits a crash we'd have caught. Over-soak = unnecessary delay. Adaptive 1d–1w (D-16). |
